# WinSuperWhisper Phase 1 - UI Implementation Plan (04-ui)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the WPF user-facing surfaces for WinSuperWhisper Phase 1 - the recording overlay (DPI-aware positioned red-square + waveform), the settings window (Display / Hotkey / Transcription / Output tabs bound to `AppConfig`), and the system tray icon (warming-up vs idle, last-transcript) - with the load-bearing DPI placement math pulled into a pure, deterministically-unit-tested function.

## The placement contract (the math everything else depends on)

`OverlayPlacement.Compute(MonitorInfo monitor, double overlayWidthDip, double overlayHeightDip)` returns `(double Left, double Top)` in **DIPs**, suitable to assign directly to a WPF `Window.Left` / `Window.Top` (WPF positions windows in DIPs).

Derivation, spelled out so the test and the implementation agree exactly:

- The monitor's working area (`WorkAreaLeft/Top/Right/Bottom`) is in **physical pixels** (that is how Win32 reports it; `Win32MonitorService` from `03-adapters` fills these from `MONITORINFO.rcWork`). The working area already **excludes** the taskbar - its bottom edge is the top of the taskbar.
- Convert the working area to DIPs by dividing by `DpiScale`.
- `workWidthDip  = (WorkAreaRight - WorkAreaLeft) / DpiScale`
- `workLeftDip   = WorkAreaLeft / DpiScale`
- `workBottomDip = WorkAreaBottom / DpiScale` (this is the top of the taskbar, in DIPs)
- The required gap is **48 physical px**, which is `48 / DpiScale` DIPs.
- `Left = workLeftDip + (workWidthDip - overlayWidthDip) / 2` (centered horizontally on the working area)
- `Top  = workBottomDip - (48 / DpiScale) - overlayHeightDip` (bottom edge 48 physical px above the taskbar)

This keeps the **physical** gap at exactly 48 px regardless of scale, because both `workBottomDip` and the `48/DpiScale` term are in DIPs and re-multiplying by `DpiScale` recovers a 48-physical-px gap. The unit test asserts this in physical space to make the intent unambiguous.

```csharp
namespace WinSuperWhisper.Tests.Overlay;

public class OverlayPlacementTests
{
    // A 1920x1080 primary monitor with a 48px-tall taskbar at the bottom.
    // Working area bottom (1032) is the top of the taskbar in PHYSICAL px.
    private static MonitorInfo Monitor(double dpiScale) => new(
        Id: "\\\\.\\DISPLAY1",
        Name: "Test Display",
        WidthPx: 1920,
        HeightPx: 1080,
        DpiScale: dpiScale,
        WorkAreaLeft: 0,
        WorkAreaTop: 0,
        WorkAreaRight: 1920,
        WorkAreaBottom: 1032,
        IsPrimary: true);

    private const double OverlayWidthDip = 320.0;
    private const double OverlayHeightDip = 56.0;

    [Theory]
    [InlineData(1.0)]
    [InlineData(1.25)]
    [InlineData(1.5)]
    public void Compute_centers_overlay_horizontally_on_working_area(double dpiScale)
    {
        var (left, _) = OverlayPlacement.Compute(Monitor(dpiScale), OverlayWidthDip, OverlayHeightDip);

        // Working area is full-width (0..1920 physical). In DIPs that is 1920/scale wide,
        // and the overlay (320 DIP) is centered in it.
        double workWidthDip = 1920.0 / dpiScale;
        double expectedLeft = (workWidthDip - OverlayWidthDip) / 2.0;
        Assert.Equal(expectedLeft, left, 6);
    }

    [Theory]
    [InlineData(1.0)]
    [InlineData(1.25)]
    [InlineData(1.5)]
    public void Compute_keeps_bottom_edge_48_physical_px_above_taskbar(double dpiScale)
    {
        var (_, top) = OverlayPlacement.Compute(Monitor(dpiScale), OverlayWidthDip, OverlayHeightDip);

        // Convert the DIP-space result back to PHYSICAL px and assert the gap is exactly 48,
        // regardless of scale. This is the load-bearing invariant.
        double overlayBottomPhysical = (top + OverlayHeightDip) * dpiScale;
        double taskbarTopPhysical = 1032.0; // WorkAreaBottom
        double gapPhysical = taskbarTopPhysical - overlayBottomPhysical;
        Assert.Equal(48.0, gapPhysical, 6);
    }

    [Fact]
    public void Compute_respects_non_zero_work_area_origin()
    {
        // A secondary monitor positioned to the right, taskbar 40px tall, at 100% DPI.
        var monitor = new MonitorInfo(
            Id: "\\\\.\\DISPLAY2",
            Name: "Secondary",
            WidthPx: 1280,
            HeightPx: 1024,
            DpiScale: 1.0,
            WorkAreaLeft: 1920,
            WorkAreaTop: 0,
            WorkAreaRight: 3200,   // 1920 + 1280
            WorkAreaBottom: 984,   // 1024 - 40
            IsPrimary: false);

        var (left, top) = OverlayPlacement.Compute(monitor, OverlayWidthDip, OverlayHeightDip);

        Assert.Equal(1920.0 + (1280.0 - 320.0) / 2.0, left, 6); // centered within the right monitor
        Assert.Equal(984.0 - 48.0 - 56.0, top, 6);              // 48px above its taskbar
    }
}
```

```csharp
using WinSuperWhisper.Core.Models;

namespace WinSuperWhisper.Core.Overlay;

/// <summary>
/// Pure, WPF-free placement math for the recording overlay. Lives in Core so it is
/// unit-testable on Linux at any DPI scale without showing a window. The WPF
/// <c>OverlayWindow</c> is a thin binding over this result.
/// </summary>
public static class OverlayPlacement
{
    /// <summary>Physical-pixel gap between the overlay's bottom edge and the taskbar (top of work area).</summary>
    public const double GapPhysicalPx = 48.0;

    /// <summary>
    /// Computes the top-left position (in device-independent pixels, WPF's Left/Top unit)
    /// for an overlay of the given DIP size on the given monitor: centered horizontally on
    /// the monitor's working area, with its bottom edge <see cref="GapPhysicalPx"/> physical
    /// pixels above the taskbar.
    /// </summary>
    public static (double Left, double Top) Compute(
        MonitorInfo monitor,
        double overlayWidthDip,
        double overlayHeightDip)
    {
        double scale = monitor.DpiScale <= 0 ? 1.0 : monitor.DpiScale;

        // Work area is reported in PHYSICAL px and already excludes the taskbar.
        double workLeftDip = monitor.WorkAreaLeft / scale;
        double workWidthDip = (monitor.WorkAreaRight - monitor.WorkAreaLeft) / scale;
        double workBottomDip = monitor.WorkAreaBottom / scale; // top of the taskbar, in DIPs

        double gapDip = GapPhysicalPx / scale;

        double left = workLeftDip + (workWidthDip - overlayWidthDip) / 2.0;
        double top = workBottomDip - gapDip - overlayHeightDip;

        return (left, top);
    }
}
```

```csharp
using System.ComponentModel;
using System.Runtime.CompilerServices;
using WinSuperWhisper.Core.Models;

namespace WinSuperWhisper.Core.Settings;

/// <summary>
/// Pure, WPF-free editable mirror of <see cref="AppConfig"/> for the Settings UI.
/// Implements <see cref="INotifyPropertyChanged"/> (which lives in System.ComponentModel,
/// not WPF) so the WIN11 SettingsWindow can data-bind to it directly. Holds Phase 1 fields only.
/// </summary>
public sealed class SettingsViewModel : INotifyPropertyChanged
{
    private string _distro = "Ubuntu";
    private string _modelPathUnc = "";
    private string _daemonScriptPathUnc = "";
    private string _language = "auto";
    private string _monitorId = "";
    private HotkeyCombo _hotkey = HotkeyCombo.Default;
    private bool _autoType = true;

    public string Distro { get => _distro; set => Set(ref _distro, value); }
    public string ModelPathUnc { get => _modelPathUnc; set => Set(ref _modelPathUnc, value); }
    public string DaemonScriptPathUnc { get => _daemonScriptPathUnc; set => Set(ref _daemonScriptPathUnc, value); }
    public string Language { get => _language; set => Set(ref _language, value); }
    public string MonitorId { get => _monitorId; set => Set(ref _monitorId, value); }
    public HotkeyCombo Hotkey { get => _hotkey; set => Set(ref _hotkey, value); }
    public bool AutoType { get => _autoType; set => Set(ref _autoType, value); }

    public static SettingsViewModel FromConfig(AppConfig config) => new()
    {
        _distro = config.Distro,
        _modelPathUnc = config.ModelPathUnc,
        _daemonScriptPathUnc = config.DaemonScriptPathUnc,
        _language = config.Language,
        _monitorId = config.MonitorId,
        _hotkey = config.Hotkey,
        _autoType = config.AutoType,
    };

    public AppConfig ToConfig() => new()
    {
        Distro = _distro,
        ModelPathUnc = _modelPathUnc,
        DaemonScriptPathUnc = _daemonScriptPathUnc,
        Language = _language,
        MonitorId = _monitorId,
        Hotkey = _hotkey,
        AutoType = _autoType,
    };

    public event PropertyChangedEventHandler? PropertyChanged;

    private void Set<T>(ref T field, T value, [CallerMemberName] string? name = null)
    {
        if (EqualityComparer<T>.Default.Equals(field, value)) return;
        field = value;
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(name));
    }
}
```

```csharp
using WinSuperWhisper.Core.Models;

namespace WinSuperWhisper.Core.Tray;

/// <summary>How the tray icon should present itself to the user.</summary>
public enum TrayPresentation { WarmingUp, Idle, Recording, Transcribing }

/// <summary>
/// Pure, WPF-free selection of tray presentation from recorder state and daemon readiness.
/// The WIN11 <c>TrayIcon</c> picks its icon/tooltip from this.
/// </summary>
public static class TrayState
{
    /// <summary>
    /// Until the daemon is READY the tray is always "warming up" - a hotkey press during model
    /// load must not look usable. Once ready, the tray mirrors the recorder state.
    /// </summary>
    public static TrayPresentation For(RecorderState recorder, bool daemonReady)
    {
        if (!daemonReady)
            return TrayPresentation.WarmingUp;

        return recorder switch
        {
            RecorderState.Recording => TrayPresentation.Recording,
            RecorderState.Transcribing => TrayPresentation.Transcribing,
            RecorderState.WarmingUp => TrayPresentation.WarmingUp,
            _ => TrayPresentation.Idle,
        };
    }

    public static string Tooltip(TrayPresentation presentation) => presentation switch
    {
        TrayPresentation.WarmingUp => "WinSuperWhisper - warming up...",
        TrayPresentation.Idle => "WinSuperWhisper - ready",
        TrayPresentation.Recording => "WinSuperWhisper - recording",
        TrayPresentation.Transcribing => "WinSuperWhisper - transcribing",
        _ => "WinSuperWhisper",
    };
}
```

```csharp
using System;
using System.Collections.ObjectModel;
using System.Windows;
using System.Windows.Media.Animation;
using WinSuperWhisper.Core.Models;
using WinSuperWhisper.Core.Overlay;

namespace WinSuperWhisper.App.Windows;

/// <summary>
/// Borderless, topmost recording overlay. Positioning is delegated entirely to the pure
/// <see cref="OverlayPlacement.Compute"/>; this class is a thin WPF binding over it plus the
/// red-square pulse, spinner, and 20-bar waveform animation.
/// </summary>
public partial class OverlayWindow : Window
{
    private const int BarCount = 20;
    private const double BarMinHeight = 2.0;
    private const double BarMaxHeight = 36.0;

    /// <summary>Bound to the waveform ItemsControl; each item is a bar height in DIPs.</summary>
    public ObservableCollection<double> Bars { get; } = new();

    private Storyboard? _pulse;
    private Storyboard? _spin;
    private bool _frozen;

    public OverlayWindow()
    {
        InitializeComponent();
        for (int i = 0; i < BarCount; i++) Bars.Add(BarMinHeight);
        WaveformBars.ItemsSource = Bars;
        BuildAnimations();
    }

    /// <summary>
    /// The DIP size WPF lays this window out at. Read from Width/Height so it stays in sync
    /// with the XAML and is the exact value handed to the pure placement function.
    /// </summary>
    public (double Width, double Height) OverlaySizeDip => (Width, Height);

    /// <summary>Positions the overlay on the given monitor (pure math), then shows it.</summary>
    public void ShowOn(MonitorInfo monitor)
    {
        var (left, top) = OverlayPlacement.Compute(monitor, Width, Height);
        Left = left;
        Top = top;
        SetState(RecorderState.Recording);
        Show();
    }

    /// <summary>Swaps the left indicator and freezes/unfreezes the bars per recorder state.</summary>
    public void SetState(RecorderState state)
    {
        switch (state)
        {
            case RecorderState.Transcribing:
                _frozen = true;
                _pulse?.Stop();
                RedSquare.Visibility = Visibility.Collapsed;
                Spinner.Visibility = Visibility.Visible;
                _spin?.Begin();
                break;

            case RecorderState.Recording:
                _frozen = false;
                _spin?.Stop();
                Spinner.Visibility = Visibility.Collapsed;
                RedSquare.Visibility = Visibility.Visible;
                _pulse?.Begin();
                break;

            default:
                _frozen = false;
                _pulse?.Stop();
                _spin?.Stop();
                break;
        }
    }

    /// <summary>Push a new amplitude (0..1) into the waveform. Ignored while frozen.</summary>
    public void UpdateLevel(AudioLevel level)
    {
        if (_frozen) return;

        // Shift bars left by one and append the newest amplitude as the rightmost bar.
        for (int i = 0; i < BarCount - 1; i++)
            Bars[i] = Bars[i + 1];

        double peak = Math.Clamp(level.Peak, 0f, 1f);
        Bars[BarCount - 1] = BarMinHeight + peak * (BarMaxHeight - BarMinHeight);
    }

    private void BuildAnimations()
    {
        // Red-square pulse: gently scale 1.0 <-> 1.12, forever.
        var scaleX = new DoubleAnimation(1.0, 1.12, TimeSpan.FromMilliseconds(600))
        {
            AutoReverse = true,
            RepeatBehavior = RepeatBehavior.Forever,
        };
        var scaleY = scaleX.Clone();
        Storyboard.SetTarget(scaleX, RedSquareScale);
        Storyboard.SetTargetProperty(scaleX, new PropertyPath("ScaleX"));
        Storyboard.SetTarget(scaleY, RedSquareScale);
        Storyboard.SetTargetProperty(scaleY, new PropertyPath("ScaleY"));
        _pulse = new Storyboard();
        _pulse.Children.Add(scaleX);
        _pulse.Children.Add(scaleY);

        // Spinner: rotate 0 -> 360, forever.
        var rotate = new DoubleAnimation(0, 360, TimeSpan.FromMilliseconds(900))
        {
            RepeatBehavior = RepeatBehavior.Forever,
        };
        Storyboard.SetTarget(rotate, SpinnerRotate);
        Storyboard.SetTargetProperty(rotate, new PropertyPath("Angle"));
        _spin = new Storyboard();
        _spin.Children.Add(rotate);
    }
}
```

```csharp
using System.Collections.ObjectModel;
using System.ComponentModel;
using System.Linq;
using System.Windows;
using WinSuperWhisper.Core.Interfaces;
using WinSuperWhisper.Core.Models;
using WinSuperWhisper.Core.Settings;

namespace WinSuperWhisper.App.Windows;

/// <summary>
/// Tabbed Phase 1 settings window. Edits a <see cref="SettingsViewModel"/> (pure, Core) and on
/// Save converts back to <see cref="AppConfig"/> and persists via the Core config store.
/// </summary>
public partial class SettingsWindow : Window
{
    // MOD_* flags per HotkeyCombo contract.
    private const uint ModAltFlag = 0x0001;
    private const uint ModControlFlag = 0x0002;
    private const uint ModShiftFlag = 0x0004;
    private const uint ModWinFlag = 0x0008;

    /// <summary>One monitor row in the Display radio list.</summary>
    public sealed class MonitorRow : INotifyPropertyChanged
    {
        private bool _isSelected;
        public string Id { get; init; } = "";
        public string Display { get; init; } = "";
        public bool IsSelected
        {
            get => _isSelected;
            set { if (_isSelected == value) return; _isSelected = value; PropertyChanged?.Invoke(this, new(nameof(IsSelected))); }
        }
        public event PropertyChangedEventHandler? PropertyChanged;
    }

    // Letter/digit virtual-key map for the key picker. VK_A=0x41.. VK_SPACE=0x20.
    private static readonly (string Label, uint Vk)[] KeyChoices =
    {
        ("Space", 0x20),
        ("A", 0x41), ("B", 0x42), ("C", 0x43), ("D", 0x44), ("E", 0x45), ("F", 0x46),
        ("G", 0x47), ("H", 0x48), ("I", 0x49), ("J", 0x4A), ("K", 0x4B), ("L", 0x4C),
        ("M", 0x4D), ("N", 0x4E), ("O", 0x4F), ("P", 0x50), ("Q", 0x51), ("R", 0x52),
        ("S", 0x53), ("T", 0x54), ("U", 0x55), ("V", 0x56), ("W", 0x57), ("X", 0x58),
        ("Y", 0x59), ("Z", 0x5A),
    };

    private static readonly (string Label, string Code)[] Languages =
    {
        ("Auto-detect", "auto"), ("English", "en"), ("Spanish", "es"), ("French", "fr"),
        ("German", "de"), ("Italian", "it"), ("Portuguese", "pt"), ("Dutch", "nl"),
        ("Russian", "ru"), ("Chinese", "zh"), ("Japanese", "ja"), ("Korean", "ko"),
    };

    private readonly IMonitorService _monitors;
    private readonly SettingsViewModel _vm;
    private readonly ObservableCollection<MonitorRow> _monitorRows = new();
    private bool _loading;

    /// <summary>The config the window was opened with; replaced with the saved value on Save.</summary>
    public AppConfig Result { get; private set; }

    /// <summary>True if the user committed changes (Save), false if cancelled/closed.</summary>
    public bool Saved { get; private set; }

    public SettingsWindow(AppConfig config, IMonitorService monitors)
    {
        InitializeComponent();
        _monitors = monitors;
        _vm = SettingsViewModel.FromConfig(config);
        Result = config;
        LoadIntoControls();
    }

    private void LoadIntoControls()
    {
        _loading = true;

        // Language dropdown.
        LanguagePicker.ItemsSource = Languages.Select(l => l.Label).ToList();
        int langIndex = System.Array.FindIndex(Languages, l => l.Code == _vm.Language);
        LanguagePicker.SelectedIndex = langIndex >= 0 ? langIndex : 0;

        // Key picker.
        KeyPicker.ItemsSource = KeyChoices.Select(k => k.Label).ToList();
        int keyIndex = System.Array.FindIndex(KeyChoices, k => k.Vk == _vm.Hotkey.VirtualKey);
        KeyPicker.SelectedIndex = keyIndex >= 0 ? keyIndex : 0;

        // Modifier checkboxes.
        ModCtrl.IsChecked = (_vm.Hotkey.Modifiers & ModControlFlag) != 0;
        ModAlt.IsChecked = (_vm.Hotkey.Modifiers & ModAltFlag) != 0;
        ModShift.IsChecked = (_vm.Hotkey.Modifiers & ModShiftFlag) != 0;
        ModWin.IsChecked = (_vm.Hotkey.Modifiers & ModWinFlag) != 0;

        // Distro + model path + auto-type.
        DistroPicker.ItemsSource = new[] { "Ubuntu", "Ubuntu-22.04", "Ubuntu-20.04", "Debian", "kali-linux" };
        DistroPicker.Text = _vm.Distro;
        ModelPathBox.Text = _vm.ModelPathUnc;
        AutoTypeToggle.IsChecked = _vm.AutoType;

        // Monitor radio list, with the configured monitor pre-selected (empty == primary).
        _monitorRows.Clear();
        var all = _monitors.GetMonitors();
        string selectedId = string.IsNullOrEmpty(_vm.MonitorId)
            ? (all.FirstOrDefault(m => m.IsPrimary)?.Id ?? all.FirstOrDefault()?.Id ?? "")
            : _vm.MonitorId;
        foreach (var m in all)
        {
            _monitorRows.Add(new MonitorRow
            {
                Id = m.Id,
                Display = $"{m.Name}  ({m.WidthPx}x{m.HeightPx})" + (m.IsPrimary ? "  [primary]" : ""),
                IsSelected = m.Id == selectedId,
            });
        }
        MonitorList.ItemsSource = _monitorRows;

        _loading = false;
    }

    private void OnHotkeyChanged(object sender, RoutedEventArgs e)
    {
        if (_loading) return;
        SyncHotkeyFromControls();
    }

    private void OnHotkeyChanged(object sender, System.Windows.Controls.SelectionChangedEventArgs e)
    {
        if (_loading) return;
        SyncHotkeyFromControls();
    }

    private void SyncHotkeyFromControls()
    {
        uint mods = 0;
        if (ModCtrl.IsChecked == true) mods |= ModControlFlag;
        if (ModAlt.IsChecked == true) mods |= ModAltFlag;
        if (ModShift.IsChecked == true) mods |= ModShiftFlag;
        if (ModWin.IsChecked == true) mods |= ModWinFlag;

        uint vk = KeyPicker.SelectedIndex >= 0 ? KeyChoices[KeyPicker.SelectedIndex].Vk : 0x20;
        _vm.Hotkey = new HotkeyCombo(mods, vk);
    }

    private void OnBrowseModelFolder(object sender, RoutedEventArgs e)
    {

        // already enables WinForms for the tray). It can browse into \\wsl$\<distro>\... UNC paths.
        using var dialog = new System.Windows.Forms.FolderBrowserDialog
        {
            Description = "Select the faster-whisper model folder (a directory, not a file)",
            UseDescriptionForTitle = true,
            ShowNewFolderButton = false,
        };
        if (!string.IsNullOrWhiteSpace(ModelPathBox.Text))
            dialog.SelectedPath = ModelPathBox.Text;

        if (dialog.ShowDialog() == System.Windows.Forms.DialogResult.OK)
            ModelPathBox.Text = dialog.SelectedPath;
    }

    private void OnSave(object sender, RoutedEventArgs e)
    {
        // Pull edited controls into the view model.
        SyncHotkeyFromControls();
        _vm.Distro = DistroPicker.Text;
        _vm.ModelPathUnc = ModelPathBox.Text;
        _vm.Language = LanguagePicker.SelectedIndex >= 0 ? Languages[LanguagePicker.SelectedIndex].Code : "auto";
        _vm.AutoType = AutoTypeToggle.IsChecked == true;

        var chosen = _monitorRows.FirstOrDefault(r => r.IsSelected);
        // Empty MonitorId means "primary"; store empty when the primary row is the chosen one.
        if (chosen is not null)
        {
            bool chosenIsPrimary = _monitors.GetMonitors().FirstOrDefault(m => m.Id == chosen.Id)?.IsPrimary == true;
            _vm.MonitorId = chosenIsPrimary ? "" : chosen.Id;
        }

        Result = _vm.ToConfig();
        AppConfigStore.Save(Result);
        Saved = true;
        Close();
    }

    private void OnCancel(object sender, RoutedEventArgs e)
    {
        Saved = false;
        Close();
    }
}
```

```csharp
using System.Threading;
using System.Windows;
using System.Windows.Threading;
using WinSuperWhisper.App.Windows;
using WinSuperWhisper.Core.Models;
using WinSuperWhisper.Core.Overlay;

namespace WinSuperWhisper.App.Tests.Windows;

public class OverlayWindowSmokeTests

    private static MonitorInfo FabricatedMonitor() => new(
        Id: "\\\\.\\DISPLAY1",
        Name: "Smoke Display",
        WidthPx: 1920,
        HeightPx: 1080,
        DpiScale: 1.0,
        WorkAreaLeft: 0,
        WorkAreaTop: 0,

    /// <summary>
    /// Runs <paramref name="body"/> on a dedicated STA thread with a live WPF Dispatcher, so a
    /// real Window can be created, shown, and torn down headlessly. The thread shuts the
    /// dispatcher down on completion.
    /// </summary>
    private static T RunOnSta<T>(Func<T> body)
    {
        T result = default!;
        Exception? captured = null;
        var thread = new Thread(() =>
        {
            try
            {
                // Ensure an Application exists for resource resolution; reuse if already created.
                if (Application.Current is null)
                    _ = new Application();
                result = body();
            }
            catch (Exception ex) { captured = ex; }
            finally { Dispatcher.CurrentDispatcher.InvokeShutdown(); }
        });
        thread.SetApartmentState(ApartmentState.STA);
        thread.Start();
        thread.Join(TimeSpan.FromSeconds(30));
        if (captured is not null) throw captured;
        return result;
    }

    [Fact]
    public void Overlay_appears_positions_per_pure_function_and_dismisses()
    {
        var monitor = FabricatedMonitor();

        var (wasVisible, left, top, expectedLeft, expectedTop, dismissed) = RunOnSta(() =>
        {
            var overlay = new OverlayWindow();
            var (w, h) = overlay.OverlaySizeDip;
            var (eLeft, eTop) = OverlayPlacement.Compute(monitor, w, h);

            overlay.ShowOn(monitor);
            DoEvents();

            bool visible = overlay.IsVisible;
            double l = overlay.Left;
            double t = overlay.Top;

            overlay.Close();
            DoEvents();
            bool gone = !overlay.IsVisible;

            return (visible, l, t, eLeft, eTop, gone);
        });

        Assert.True(wasVisible, "overlay should be visible after ShowOn");
        Assert.Equal(expectedLeft, left, 3);   // positioned exactly per the pure placement math
        Assert.Equal(expectedTop, top, 3);
        Assert.True(dismissed, "overlay should be dismissed after Close");
    }

    /// <summary>Pumps queued dispatcher work so layout/show/close take effect synchronously.</summary>
    private static void DoEvents()
    {
        var frame = new DispatcherFrame();
        Dispatcher.CurrentDispatcher.BeginInvoke(
            DispatcherPriority.Background,
            new DispatcherOperationCallback(f => { ((DispatcherFrame)f).Continue = false; return null; }),
            frame);
        Dispatcher.PushFrame(frame);
    }
}
```

```csharp
using System;
using System.Drawing;
using System.Windows.Forms;
using WinSuperWhisper.Core.Models;
using WinSuperWhisper.Core.Tray;

namespace WinSuperWhisper.App.Tray;

/// <summary>
/// System tray icon hosted via the WinForms <see cref="NotifyIcon"/> from the WPF app.
/// Presentation (icon tint + tooltip) is driven by the pure <see cref="Core.Tray.TrayState"/>;
/// the menu exposes Settings / last-transcript / Exit. The App layer (05-movein) subscribes to
/// the events to open settings and shut down.
/// </summary>
public sealed class TrayIcon : IDisposable
{
    private readonly NotifyIcon _icon;
    private readonly ToolStripMenuItem _settingsItem;
    private readonly ToolStripMenuItem _transcriptItem;
    private readonly ToolStripMenuItem _exitItem;

    private string _lastTranscript = "";

    /// <summary>Raised when the user clicks Settings.</summary>
    public event EventHandler? SettingsRequested;

    /// <summary>Raised when the user clicks Exit.</summary>
    public event EventHandler? ExitRequested;

    public TrayIcon()
    {
        _settingsItem = new ToolStripMenuItem("Settings", null, (_, _) => SettingsRequested?.Invoke(this, EventArgs.Empty));
        _transcriptItem = new ToolStripMenuItem("(no transcript yet)") { Enabled = false };
        _exitItem = new ToolStripMenuItem("Exit", null, (_, _) => ExitRequested?.Invoke(this, EventArgs.Empty));

        var menu = new ContextMenuStrip();
        menu.Items.Add(_settingsItem);
        menu.Items.Add(_transcriptItem);
        menu.Items.Add(new ToolStripSeparator());
        menu.Items.Add(_exitItem);

        _icon = new NotifyIcon
        {
            Visible = true,
            ContextMenuStrip = menu,
            Text = TrayState.Tooltip(TrayPresentation.WarmingUp),
            Icon = BuildIcon(TrayPresentation.WarmingUp),
        };

        // Left-click shows the last transcript as a balloon tip.
        _icon.MouseClick += (_, e) =>
        {
            if (e.Button == MouseButtons.Left) ShowLastTranscript();
        };
    }

    /// <summary>
    /// Updates the tray to reflect the current recorder state and daemon readiness, using the
    /// pure presentation mapping. Tooltip is capped at 63 chars (NotifyIcon.Text limit).
    /// </summary>
    public void UpdateState(RecorderState recorder, bool daemonReady)
    {
        var presentation = TrayState.For(recorder, daemonReady);
        string tip = TrayState.Tooltip(presentation);
        _icon.Text = tip.Length > 63 ? tip[..63] : tip;
        _icon.Icon?.Dispose();
        _icon.Icon = BuildIcon(presentation);
    }

    /// <summary>Records the latest transcript and reflects it in the right-click menu label.</summary>
    public void SetLastTranscript(string transcript)
    {
        _lastTranscript = transcript ?? "";
        string label = string.IsNullOrWhiteSpace(_lastTranscript)
            ? "(no transcript yet)"
            : "Last: " + Truncate(_lastTranscript, 40);
        _transcriptItem.Text = label;
    }

    private void ShowLastTranscript()
    {
        if (string.IsNullOrWhiteSpace(_lastTranscript))
        {
            _icon.ShowBalloonTip(2000, "WinSuperWhisper", "No transcript yet.", ToolTipIcon.Info);
            return;
        }
        _icon.ShowBalloonTip(4000, "Last transcript", Truncate(_lastTranscript, 255), ToolTipIcon.Info);
    }

    /// <summary>
    /// Builds a 16x16 tray icon tinted by presentation so warming-up vs idle vs active is
    /// distinguishable without shipping image assets: amber (warming up), green (idle),
    /// red (recording), blue (transcribing).
    /// </summary>
    private static Icon BuildIcon(TrayPresentation presentation)
    {
        Color color = presentation switch
        {
            TrayPresentation.WarmingUp => Color.FromArgb(0xE0, 0xA0, 0x30),    // amber
            TrayPresentation.Idle => Color.FromArgb(0x3C, 0xB0, 0x4B),         // green
            TrayPresentation.Recording => Color.FromArgb(0xE0, 0x3A, 0x3A),    // red
            TrayPresentation.Transcribing => Color.FromArgb(0x4A, 0x80, 0xE0), // blue
            _ => Color.Gray,
        };

        using var bmp = new Bitmap(16, 16);
        using (var g = Graphics.FromImage(bmp))
        {
            g.SmoothingMode = System.Drawing.Drawing2D.SmoothingMode.AntiAlias;
            g.Clear(Color.Transparent);
            using var brush = new SolidBrush(color);
            g.FillEllipse(brush, 2, 2, 12, 12);
        }
        // Clone into an Icon. The HICON must be destroyed by the caller; NotifyIcon owns it via Dispose.
        return Icon.FromHandle(bmp.GetHicon());
    }

    private static string Truncate(string s, int max) =>
        s.Length <= max ? s : s[..max] + "...";

    public void Dispose()
    {
        _icon.Visible = false;
        _icon.Icon?.Dispose();
        _icon.Dispose();
    }
}
```
